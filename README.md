## Info

L'app è fatta con Nextjs

## Avvio in locale

**Assicurarsi di avere NodeJS 24 installato**

1. Clonare la repo
2. Eseguire ``npm i`` nella cartella della repo

Avvia il webserver con

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Apri [http://localhost:3000](http://localhost:3000) sul browser e l'app sarà in funzione
